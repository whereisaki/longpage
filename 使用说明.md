# 长图生成器 · 使用说明

## 📦 包含工具

| 文件 | 功能 |
|------|------|
| **index.html** | 主站首页，选择使用哪个工具 |
| **封面生成器.html** | 制作封面卡片（1080×1440） |
| **长文排版器.html** | 将 Markdown 长文自动分页排版为图片 |

## 🚀 使用方式

1. 双击 `index.html` 打开主站首页
2. 选择要使用的工具（封面生成器或长文排版器）
3. 在左侧面板输入内容
4. 右侧实时预览
5. 点击「导出」按钮保存为 PNG 图片

> ⚠️ 请不要移动或删除 `fonts/` 和 `lib/` 文件夹，它们是工具运行所需的资源

## 🎨 功能特色

### 封面生成器
- 支持主标题、副标题、装饰文字编辑
- 支持上传背景图片
- 自定义字体选择（支持系统字体和本地字体）
- 自定义背景色、标题色、副标题色

### 长文排版器
- 粘贴 Markdown 自动渲染排版
- **自动分页**——内容再长也不会溢出
- 自定义页眉/页脚文字（填入你自己的品牌名）
- 可调节字号、行间距、段落间距
- 自定义字体选择（支持系统字体和本地字体检测）
- 自定义背景色、文字色
- 支持 Markdown 图片语法
- 支持批量导出为 PNG + ZIP 打包下载

## 🖌️ 自定义品牌

在「长文排版器」的正文排版页顶部：
- **页眉文字**：显示在每页右上角（默认：长图生成器）
- **页脚署名**：显示在每页左下角（默认：仅个人使用）

留空则使用默认文字。

## 📝 Markdown 图片语法

在长文排版器中，使用以下语法插入图片：

```
![图片描述](图片URL)
```

示例：
```
![我的照片](./images/photo.jpg)
![网络图片](https://example.com/image.png)
```

图片会自动适应宽度并保持比例。

## ❓ 常见问题

**Q: 字体显示不正常？**
A: 确保 `fonts/` 文件夹和 HTML 文件在同一目录下，不要单独移动 HTML 文件。

**Q: 图片不显示？**
A: 确保使用正确的 Markdown 图片语法，图片 URL 可以是本地路径或网络地址。

**Q: 导出的图片模糊？**
A: 工具默认按 3 倍分辨率导出（3240×4320），清晰度足够使用。

**Q: 支持什么浏览器？**
A: 推荐 Chrome / Edge。Safari 可用但导出方式略有不同。

**Q: 如何检测本地字体？**
A: 在长文排版器中点击「检测本地字体」按钮，系统会自动扫描并显示可用的本地字体。
