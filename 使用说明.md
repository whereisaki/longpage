# 小红书图片生成器 · 使用说明

## 📦 包含工具

| 文件 | 功能 |
|------|------|
| **封面生成器.html** | 制作小红书封面卡片（1080×1440） |
| **长文排版器.html** | 将 Markdown 长文自动分页排版为图片 |

## 🚀 使用方式

1. 双击 `.html` 文件，用浏览器（推荐 Chrome）打开
2. 在左侧面板输入内容
3. 右侧实时预览
4. 点击「导出」按钮保存为 PNG 图片

> ⚠️ 请不要移动或删除 `fonts/` 和 `lib/` 文件夹，它们是工具运行所需的资源

## 🎨 功能特色

### 封面生成器
- 支持标题、副标题、目录编辑
- 支持上传人物图片（最多 4 张）
- 可调整图片缩放和圆角
- 多种配色方案一键切换

### 长文排版器
- 粘贴 Markdown 自动渲染排版
- **自动分页**——内容再长也不会溢出
- 自定义页眉/页脚文字（填入你自己的品牌名）
- 可调节字号
- 多种配色方案
- 支持批量导出为 PNG + ZIP 打包下载

## 🖌️ 自定义品牌

在「长文排版器」的正文排版标签页顶部，有两个输入框：
- **页眉文字**：显示在每页右上角（例：你的专栏名）
- **页脚署名**：显示在每页左下角（例：你的账号名）

留空则使用默认文字。

## ❓ 常见问题

**Q: 字体显示不正常？**
A: 确保 `fonts/` 文件夹和 HTML 文件在同一目录下，不要单独移动 HTML 文件。

**Q: 导出的图片模糊？**
A: 工具默认按 3 倍分辨率导出（3240×4320），清晰度足够小红书使用。

**Q: 支持什么浏览器？**
A: 推荐 Chrome / Edge。Safari 可用但导出方式略有不同。
